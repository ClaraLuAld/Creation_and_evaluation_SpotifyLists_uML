# Creation_and_evaluation_SpotifyLists_uML
This repository contains a unsupervised Machine Learning project. It was developed as part of a group work of a Data Science Bootcamp. process to create lists of songs from a Spotify dataset. It also contains the final presentation that took place on April 2025.

This repository contains two python Colab notebooks with the unsupervised ML process to create lists of songs from a Spotify dataset provided by the school. It also contains the final presentation
